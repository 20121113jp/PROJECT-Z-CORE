# Publishing

This tree is intended to be publishable as a GitHub repository.

## Checklist

1. Review the generated files.
2. Confirm no leaked proprietary source was copied in.
3. Run `npm test`.
4. Create a new GitHub repository.
5. Push this tree as its own repository root.

## Growth path

- Add adapter implementations under `adapters/`.
- Clone or vendor official public upstreams under `vendors/`.
- Keep the capability matrix honest as adapters become real.
